# SignalSync Website

Professional NextGen TV 3.0 and MIMO antenna installation services website.

## Features
- Responsive design
- Custom web components
- Smooth scrolling
- Interactive UI elements
- Dark mode support

## Setup
1. Clone the repository
2. Open index.html in a browser
3. For development, use a local server

## Technology Stack
- HTML5
- CSS3
- JavaScript (Custom Web Components)
- TailwindCSS
